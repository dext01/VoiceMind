# VoiceMind

VoiceMind — система голосовых заметок с офлайн-транскрипцией и AI-анализом. Состоит из двух частей: носимого устройства на ESP32 с микрофоном и Android-приложения.

Устройство захватывает речь через I2S-микрофон и передаёт аудио на телефон по Bluetooth. Приложение транскрибирует запись с помощью Whisper, определяет говорящих и при наличии LLM-модели выделяет ключевые мысли из текста. Всё работает без интернета.

---

## Установка приложения

1. Скачайте APK из раздела [Releases](https://github.com/dext01/VoiceMind/releases)
2. Установите на Android 8.0+ (разрешите установку из неизвестных источников)
3. При первом запуске приложение автоматически скачает модель распознавания речи (~75 МБ)
4. Опционально: скачать модель анализа текста (~900 МБ) — выделяет ключевые мысли из записей

После этого приложение работает полностью офлайн.

---

## Что умеет

- Запись с внешнего микрофона ESP32 через Bluetooth
- Офлайн-транскрипция через Whisper
- Определение нескольких говорящих (диаризация)
- Выделение ключевых мыслей через локальную LLM (Qwen 2.5 1.5B)
- История записей по дням

## Железо (ESP32-устройство)

Приложение подключается к устройству `VoiceMind-01` по Bluetooth SPP. Устройство собирается на NodeMCU ESP32 с микрофоном ICS-43434.

Схема подключения микрофона:

| Микрофон | ESP32   |
|----------|---------|
| WS       | GPIO 26 |
| SCK      | GPIO 25 |
| SD       | GPIO 22 |
| VDD      | 3.3V    |
| GND      | GND     |
| L/R      | GND     |

Прошивка в папке `src/`, сборка через PlatformIO:

```
pio run --target upload
```

Спарьте устройство с телефоном в настройках Bluetooth до запуска приложения.

## Улучшенное качество распознавания (опционально)

По умолчанию используется Whisper tiny (75 МБ). Если нужно лучшее качество, скопируйте более тяжёлую модель через adb:

```
adb push ggml-small.bin /data/data/com.example.voicemind/files/
```

Доступные модели: [huggingface.co/ggerganov/whisper.cpp](https://huggingface.co/ggerganov/whisper.cpp) — файлы `ggml-*.bin`.

---

## Диаризация говорящих (опционально)

Для точного определения говорящих нужна ONNX-модель ECAPA-TDNN. Без неё диаризация работает по паузам в речи.

```
adb push speaker_model.onnx /data/data/com.example.voicemind/files/
```
