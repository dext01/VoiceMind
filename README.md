# VoiceMind

VoiceMind — приложение для голосовых заметок с офлайн-транскрипцией и AI-анализом.

Запись ведётся с внешнего микрофона на ESP32 через Bluetooth или с микрофона телефона. Приложение расшифровывает речь, определяет говорящих и выделяет ключевые мысли. Всё работает без интернета.

---

## Установка

1. Скачайте `VoiceMind.apk` из раздела [Releases](https://github.com/dext01/VoiceMind/releases)
2. Установите на Android 8.0 или новее (нужно разрешить установку из неизвестных источников)
3. При первом запуске приложение скачает модели для распознавания речи и анализа текста — потребуется около 1 ГБ свободного места и несколько минут на загрузку
4. После этого интернет не нужен

---

## Железо

Для записи через внешний микрофон нужно устройство на ESP32 с микрофоном ICS-43434. Устройство появляется в Bluetooth как `VoiceMind-01` — спарьте его с телефоном в настройках Bluetooth до запуска приложения.

Если устройства нет, приложение пишет с микрофона телефона.
